== Description ==

Tasti Rapidi è un semplice plug-in di WordPress che ti consente di poter salvare, lavorare e navigare sul tuo sito WordPress tramite comandi rapidi da tastiera semplicemente premendo i tasti. Il modo più semplice e veloce per risparmiare sui tempi di click con il mouse nel tuo sito.

= Tasti Rapidi =

Dopo l'installazione del plugin, premi il tasto "?" per poter accedere alla lista delle funzionalità, di seguito riportate


`?` → Apertura popup area di aiuto/help
`Ctrl+s` → Salvataggio rapido articolo/pagina
`m` → Modifica la pagina / articolo corrente da frontend
`r` →  Ricarica/refresh della pagina
`v` e poi premendo `a` → Visualizza tutti gli Articoli in admin
`v` e poi premendo `n`→ Crea Nuovo articolo in admin
`Shift + v` e poi successivamente `p` → Visualizza tutte le Pagine in admin
`Shift + v` e poi successivamente `n` → Crea Nuova pagina in admin
`v` e poi premendo `h` → Vai alla Homepage del tuo frontend
`v` e poi premendo `l` → Vai alla pagina di Login, disconnettendoti
`v` e poi premendo `d` → Vai alla tua Dashboard in admin
`v` e poi premendo `c` → Vai ai Commenti in admin
`v` e poi premendo `t` → Vai ai Temi in admin
`v` e poi premendo `p` → Vai ai Plugin in admin
`v` e poi premendo `u` → Vai agli Utenti in admin
`v` e poi premendo `i` → Vai alle Impostazioni in admin


== Installation ==

Puoi caricare il file zip da Plugin/Aggiungi Nuovo, dal pannello di amministrazione del tuo wordpress 
oppure
Estrai il file zip e rilascia il contenuto nella directory wp-content / plugins / della tua installazione di WordPress, quindi attiva la pagina Plugin da Plugins.
